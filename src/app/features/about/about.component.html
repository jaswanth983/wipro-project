About This Application

This is a simple yet powerful **Inventory Management System** designed to help users efficiently manage their product inventory. The application enables users to **view, add, edit, and delete products** with ease. It is built using **Angular** for the front-end and utilizes a **JSON server** as a lightweight back-end.

With this application, businesses can keep track of their stock, ensuring they never run out of essential products. The user-friendly interface makes managing inventory a seamless experience.
